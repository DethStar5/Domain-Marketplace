# 🌐 Domain Marketplace (Soroban Smart Contract)

## 📌 Project Description
Domain Marketplace is a decentralized application (dApp) built on the Stellar network using Soroban smart contracts. It allows users to register, list, and trade domain names securely without relying on a centralized authority.

---

## ⚙️ What it does
- Users can register unique domain names on-chain  
- Domain owners can list their domains for sale  
- Other users can purchase available domains  
- Ownership is transparently stored on the blockchain  

---

## 🚀 Features
- 🔐 Decentralized domain ownership  
- 🏷️ Domain listing with price  
- 💰 Peer-to-peer domain transfer  
- 📜 On-chain storage using Soroban  
- ⚡ Fast and low-cost transactions (Stellar network)  

---

## 🛠️ Tech Stack
- Soroban (Stellar Smart Contracts)  
- Rust  
- Stellar SDK  

---

## 📂 Smart Contract Functions

### 1. Register Domain
Registers a new domain with an owner.

### 2. List for Sale
Allows the owner to list their domain with a price.

### 3. Buy Domain
Transfers ownership to the buyer (basic version).

### 4. Get Domain
Fetch domain details like owner, price, and sale status.

---

## ⚠️ Limitations (Basic Version)
- No real token payment handling yet  
- No auction system  
- No expiration system  
- No domain validation
